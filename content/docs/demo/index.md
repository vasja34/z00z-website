# Demo Pages Overview

Use this section to inspect the first file-backed routes:

- [Theme Lab](/docs/demo/theme-lab)
- [MD Plugins Test](/docs/demo/md-plugins-test)
- [Code Highlight Test](/docs/demo/code-highlight-test)
- [ZUZ Rust Engineer 2](/docs/demo/zuz-rust-engineer-2)

Each page exercises a different concern:

- theme switching through curated `daisyUI` presets
- code highlighting preset switching through the shared header theme panel
- advanced Markdown parsing through `mdit-plugins`
- syntax highlighting inside a raw `.html` document
- style-isolated rendering for a complex ready-made HTML export
